# Mission Control

Executive command center for all operations. Real-time dashboard for projects, systems, and tasks.

## Features

- **Projects Dashboard**: Track all active projects with health indicators, metrics, and progress
- **Systems Monitoring**: Monitor bots, cron jobs, and deployments in real-time  
- **Tasks Pipeline**: View all tasks with priorities, status, and deadlines
- **Clean Design**: Professional, Apple-worthy interface with no fluff

## Data Sources

- Trading bots (Kalshi Weather Edge, Crypto Bot)
- OpenClaw cron jobs
- Render deployments
- Notion projects and tasks

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Server-side rendering

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed to Render with automatic builds on push to master.

## Decision Support

Mission Control provides strategic insights:
- ROI analysis for each project
- Resource allocation recommendations
- Scale/maintain/kill recommendations based on performance
